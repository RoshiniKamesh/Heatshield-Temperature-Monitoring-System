# HeatShield: Smart Temperature Monitoring System

## Project Overview
HeatShield is a smart temperature monitoring system developed using Arduino Uno and a TMP36 temperature sensor to help prevent heat stroke among industrial workers.

## Components Used
- Arduino Uno
- TMP36 Temperature Sensor
- LED
- Piezo Buzzer
- Breadboard
- Jumper Wires

## Features
- Real-time temperature monitoring
- High temperature warning
- Heat stroke alert system
- Visual and audible notifications

## Applications
- Construction worker safety
- Industrial safety helmets
- Agricultural worker protection
- Outdoor workforce monitoring
